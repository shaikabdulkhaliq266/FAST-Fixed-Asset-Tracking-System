# FAST — Fixed Asset Tracking System

A full-stack desktop ERP module built with Java 11, JavaFX and SQLite.

## Features
- Real-time Asset Tracking Dashboard
- Employee & Assignment Management  
- Straight-Line Depreciation Calculator
- Live SQL Query Execution
- Automated Professional PDF Report Generation
- Export to .txt

## Tech Stack
- Java 11 | JavaFX | SQLite | JDBC

## Architecture
MVC Pattern | OOP Principles | Service Layer
